### To run the application:

- Install anaconda or miniconda that is compatible with python 3.10 <br>

For MacOS and Linux:
In the top folder of this application <br>
- run: `conda env create --file=./backend/environment.yml` <br>
- run: `sh bootstrap.sh`

For windows:
In the top folder of this application <br>
- run: `conda env create --file=.\backend\environment.yml` <br>
- run: `bootstrap.bat`
